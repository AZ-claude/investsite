# investsite

個人投資家向けの指標インサイトサイト。株価そのものではなく「**どの指標がどれだけ株価に効くのか(寄与度)**」を軸に据え、
日米市場(積立ETF=米国、個別株=日本)を対象に無料データで日次更新する静的サイト。閲覧者が「なんとなく売買」から
「基準を持った売買」へ移行できることを目的とし、投資助言ではなく情報提供に徹する。

詳細は `docs/01-strategy.md` ほか `docs/` 配下を参照。

## ディレクトリ構成

- `site/` — Astro製の静的サイト本体
- `pipeline/` — データ取得・指標計算のPythonパイプライン
- `data/` — 日次スナップショットJSON(蓄積データ)
- `docs/` — 戦略・調査・設計ドキュメント

## ローカル起動手順(site/)

```bash
cd site
npm install
npm run dev     # 開発サーバー起動
npm run build   # 本番ビルド
```
