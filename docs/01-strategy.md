# 01. 戦略 — 指標インサイトサイト

最終更新: 2026-07-10 / 作成: 戦略マネージャ

## 1. 提供価値(差別化の核)

既存サイト(Yahoo!ファイナンス、株探、バフェットコード、Finviz等)は「指標の現在値」を見せる。
本サイトは **「その指標は歴史的にどれだけリターンに効いたか(寄与度)」と現在値をセットで見せる** ことで差別化する。

例: 「PBR 0.8倍」とだけ出すのではなく、
「PBR最低位クォンタイルは日本株で年率+X%の超過リターン実績(検証期間YYYY-YYYY)。本日この条件に該当する銘柄はN件」と出す。

## 2. ユーザーストーリー(ペルソナ=投資歴7-8年・基準なし層)

1. 毎朝サイトを開くと「本日のダッシュボード」で日米市場の主要指標の現在地(割高/割安の歴史的位置)が1画面で分かる
2. 気になった指標(例: モメンタム)をクリックすると、その指標の解説・寄与度エビデンス・過去推移・該当銘柄リストが個別ページで見れる
3. 継続閲覧により「自分はどの指標を基準に売買するか」を自分で決められるようになる
